# 📚 知识库可视化浏览器

独立的知识库搜索和浏览界面，用于查看和检索阿尔茨海默病相关的医学知识。

## ✨ 功能特点

- **智能搜索**：基于语义相似度的知识检索
- **句子高亮**：自动标记最相关的句子
- **相关度评分**：显示每条结果的匹配度
- **完整内容查看**：可展开查看文档全文
- **统计信息**：显示知识库文档和片段数量
- **示例查询**：提供常见查询示例快速开始

## 🚀 快速启动

### 方法1：使用启动脚本（推荐）

```bash
cd kb_viewer
./start_kb_viewer.sh
```

### 方法2：直接运行

```bash
cd kb_viewer
python3 -m streamlit run app.py --server.port 8504
```

## 🌐 访问地址

启动后在浏览器访问：
```
http://localhost:8504
```

## 📖 使用方法

### 1. 搜索知识

在搜索框输入查询内容，例如：
- "阿尔茨海默病的早期症状"
- "定向力评估方法"
- "MMSE量表"
- "记忆力测试"

### 2. 查看结果

- **相关度徽章**：显示匹配分数（绿色=高相关，黄色=中等，灰色=低相关）
- **高亮句子**：自动提取最相关的3句话
- **展开查看**：点击"查看完整内容"查看全文

### 3. 调整设置

在左侧边栏可以：
- 调整返回结果数量（1-20条）
- 查看知识库统计信息

### 4. 快速示例

点击页面底部的示例查询按钮快速开始。

## 🔧 技术细节

### 核心组件

- **向量检索**：使用 `BAAI/bge-m3` 嵌入模型
- **重排序**：使用 `BAAI/bge-reranker-base` 句子过滤器
- **向量数据库**：ChromaDB
- **界面框架**：Streamlit

### 数据来源

知识库位于 `kb/chunks_semantic_per_file/`，包含18篇阿尔茨海默病相关的中文医学文献。

## 📊 界面说明

### 侧边栏

- **知识库统计**：显示文档数量和知识片段总数
- **搜索设置**：调整返回结果数量
- **使用说明**：简要使用指引

### 主界面

- **搜索框**：输入查询内容
- **搜索结果**：按相关度排序显示
- **示例查询**：常见查询快速按钮

### 结果卡片

每个结果包含：
- 排名和相关度评分
- 来源文档名称
- 高亮的相关句子（含分数）
- 可展开的完整内容
- 文档元数据

## 💡 使用技巧

1. **精确查询**：使用具体的医学术语效果更好
2. **组合关键词**：如"阿尔茨海默病 + 定向力 + 评估"
3. **调整结果数**：复杂查询可增加返回数量
4. **查看元数据**：了解文档来源和上下文

| 功能 | 知识库浏览器 | 主应用 (app.py) |
|------|--------------|----------------|
| **用途** | 浏览和搜索知识 | 患者认知评估 |
| **端口** | 8504 | 8503 |
| **用户** | 医生/研究人员 | 评估系统 |
| **功能** | 知识检索 | AI对话评估 |

可以**同时运行**两个应用，互不影响。

## 🔍 常见问题

### Q: 搜索没有结果？
A: 
- 检查知识库是否已构建（`kb/.chroma_semantic/`目录）
- 尝试更简单的关键词
- 确保输入的是中文

### Q: 相关度很低？
A: 
- 使用更准确的医学术语
- 尝试不同的关键词组合
- 增加返回结果数量查看更多

### Q: 如何更新知识库？
A: 
1. 添加新文档到 `kb/chunks_semantic_per_file/`
2. 运行 `scripts/index_jsonl_to_chroma.py` 重建索引
3. 重启知识库浏览器
